Como percorrer este curso

Todos os pontos do curso estão disponíveis desde o início. A jornada sugere uma ordem de estudo para facilitar sua experiência, mas você pode acessar os conteúdos conforme sua necessidade.

  • Comece pelo Sobre o Curso para entender a proposta.
  • Siga a ordem sugerida para construir repertório progressivamente.
  • Use a Rede de apoio sempre que tiver dúvidas.
  • Consulte as Ferramentas para aprofundar conceitos e práticas.
